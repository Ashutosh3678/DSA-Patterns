# DSA Patterns

A structured collection of Data Structures and Algorithms problems organized by common problem-solving patterns.

This repository focuses on consistency, clarity, and pattern-based practice for coding interviews and competitive programming.

---

## Patterns Included

- Sliding Window  
- Two Pointers  
- Prefix Sum  
- Binary Search  
- Hashing  
- Greedy  
- Backtracking  
- Dynamic Programming  
And many more topics to go
